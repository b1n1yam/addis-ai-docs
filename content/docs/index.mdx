## Developer Experience

We prioritize simplicity. Since we are API-first, you can use any language that supports HTTP.

### The "Hello World"

Copy this code to generate your first Amharic response. You don't need to install a library—just use standard requests.

<Tabs items={['JavaScript', 'Python', 'cURL']}>
  <Tab value="JavaScript">
    ```javascript
    // No SDK required. Just use fetch.
    const API_KEY = "YOUR_API_KEY";

    async function chat() {
      const response = await fetch("https://api.addisassistant.com/api/v1/chat_generate", {
        method: "POST",
        headers: {
          "Content-Type": "application/json",
          "X-API-Key": API_KEY,
        },
        body: JSON.stringify({
          model: "amharic-gpt-4",
          prompt: "ሰላም፣ ስለ አድዋ ጦርነት በአጭሩ ንገረኝ።", // "Tell me briefly about Adwa"
          target_language: "am"
        }),
      });

      const data = await response.json();
      console.log(data.response_text);
    }

    chat();
    ```
  </Tab>
  <Tab value="Python">
    ```python
    import requests

    url = "https://api.addisassistant.com/api/v1/chat_generate"
    headers = {
        "Content-Type": "application/json",
        "X-API-Key": "YOUR_API_KEY"
    }
    payload = {
        "model": "amharic-gpt-4",
        "prompt": "ሰላም፣ ስለ አድዋ ጦርነት በአጭሩ ንገረኝ።",
        "target_language": "am"
    }

    response = requests.post(url, headers=headers, json=payload)
    print(response.json()['response_text'])
    ```
  </Tab>
  <Tab value="cURL">
    ```bash
    curl -X POST https://api.addisassistant.com/api/v1/chat_generate \
      -H "X-API-Key: YOUR_API_KEY" \
      -H "Content-Type: application/json" \
      -d '{
        "model": "amharic-gpt-4",
        "prompt": "ሰላም፣ ስለ አድዋ ጦርነት በአጭሩ ንገረኝ።",
        "target_language": "am"
      }'
    ```
  </Tab>
</Tabs>

### Getting your Keys
To run the code above, you need an API Key.
1. Log in to the [Platform Dashboard](https://platform.addisassistant.com).
2. Go to **Settings > API Keys**.
3. Click **Create New Key**.
